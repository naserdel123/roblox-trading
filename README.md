# 🎮 تريدات روبلوكس (Roblox Trades)

منصة تداول عربية متكاملة لعبة روبلوكس مع تصميم عصري وأنميشن احترافي.

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ المميزات الرئيسية

### 🔐 نظام المصادقة
- تسجيل حساب جديد مع التحقق من البيانات
- تسجيل دخول آمن مع خيار "تذكرني"
- حفظ البيانات في LocalStorage (لا يحتاج سيرفر)
- تشفير كلمات المرور (Base64)

### 📱 واجهة المستخدم
- ✅ تصميم زجاجي Glassmorphism عصري
- ✅ أنميشن احترافي وسلس (CSS3 + JavaScript)
- ✅ تأثيرات 3D تفاعلية على البطاقات
- ✅ خلفيات متحركة مع جسيمات عائمة
- ✅ تصميم متجاوب (Responsive) لجميع الأجهزة

### 📝 نظام المنشورات
- نشر تريدات جديدة مع صور
- دعم فئات متعددة (Limited, Robux, Game Pass, Avatar)
- إضافة روابط TikTok للتواصل
- نظام إعجابات على المنشورات
- فلترة وترتيب المنشورات

### 🎨 التأثيرات البصرية
- تأثيرات hover متقدمة على الأزرار
- حركات انتقالية سلسة بين الصفحات
- تأثيرات loading احترافية
- Toast notifications متحركة
- تأثيرات shine و glow على العناصر التفاعلية
